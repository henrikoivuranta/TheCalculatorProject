About this website!

This website was created and maintained by a young student from Finland. Goal of this project is to have as many different calculators as possible in a single place. This project is a part of my personal learning in coding and web design.


If you have any suggestions or questions, please send me an email to this address thecalculatorproject@gmail.com. I'm working on adding a contact form system to this website to make contacting easier.
